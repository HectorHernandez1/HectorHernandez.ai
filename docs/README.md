## Resume Guidelines for Backend Engineers

### Content to Include

1. **Contact Information**
   - Name
   - Location (City, State)
   - Email
   - Phone (optional)
   - LinkedIn
   - GitHub

2. **Professional Summary**
   - 2-3 sentences highlighting your expertise
   - Focus on backend systems, data engineering, and scale
   - Mention years of experience and key technologies

3. **Technical Skills**
   - Backend Languages: Python, C#, Java, C++
   - Databases: SQL Server, PostgreSQL, Oracle
   - Data Engineering: Airflow, ETL, Docker
   - Tools & Frameworks: FastAPI, Flask, .NET
   - Other: Git, Agile, CI/CD

4. **Professional Experience**
   - Start with most recent position
   - Use bullet points with quantifiable achievements
   - Focus on impact and results, not just responsibilities
   - Highlight: performance improvements, cost savings, scale

5. **Education**
   - MS Computer Science - Arizona State University (Expected 2026, GPA 3.75)
   - BS Chemical Engineering - Arizona State University (2015)

6. **Projects** (Optional)
   - Include if you have 2-3 standout projects
   - Link to GitHub repositories
   - Highlight technologies and impact

### Format Guidelines

- **Length**: 1-2 pages (1 page preferred for 5 years experience)
- **Format**: PDF only
- **Font**: Professional sans-serif (Arial, Calibri, Helvetica)
- **Margins**: 0.5" - 1" on all sides
- **File Size**: Under 2MB
- **Filename**: `Hector_Hernandez_Resume.pdf` (no spaces, professional naming)

### ATS Optimization

Make sure your resume is ATS (Applicant Tracking System) friendly:
- Use standard section headings
- Avoid tables, images, or complex formatting
- Use keywords from job descriptions
- Include both acronyms and full terms (e.g., "ETL (Extract, Transform, Load)")
- Save as PDF/A format if possible

### Tools for Creating Resume

- **Google Docs** - Free, collaborative, easy to export
- **Microsoft Word** - Professional standard
- **LaTeX** - For technical/academic resumes (Overleaf.com)
- **Canva** - Beautiful templates (but watch ATS compatibility)
- **Resume.io** - Modern resume builder
- **NovoResume** - ATS-friendly templates
