## Image Guidelines

- **Format**: JPG, PNG, or WebP (WebP preferred for performance)
- **Dimensions**: 800x600px or similar 4:3 ratio
- **File Size**: Optimize to <200KB per image
- **Quality**: High enough to look professional, compressed for web performance

## Temporary Placeholder

The website includes JavaScript that automatically generates gradient placeholder images if the files are missing. This allows you to see the layout while you prepare your actual project screenshots.

## Tools for Creating Images

- **Screenshots**: Use macOS Screenshot (Cmd+Shift+4) or Snipping Tool (Windows)
- **Diagrams**: draw.io (diagrams.net), Lucidchart, or Figma
- **Optimization**: TinyPNG, ImageOptim, or online tools like Squoosh
- **Mockups**: Use browser dev tools to create clean screenshots without personal data
